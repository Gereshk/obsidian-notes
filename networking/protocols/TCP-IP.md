
# Transport Protocol

## 3 Way Handshake:
Sets up connection b/w two machines:
- ex: Bob calls Alice, the call connects but someone has to say  "hello?" to know they've successfully connected.
- #ACK-flag: the "hello"
- #SYN/ACK-flag: sent by target machine once it receives ACK

> #RFC-793
