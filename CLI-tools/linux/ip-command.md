
ip - show / manipulate routing, network devices, interfaces and tunnels
# Usage:
```
ip [OPTIONS] OBJECT {COMMAND | help}
```

## Useful options:
- ``ip link list``
	- ``link`` = network device
	- ``list`` = show (all objects / devices)
- ``ip address show``
	- ``address`` = protocol (IP or IPv6) address on a device
	- ``show`` = same as list
